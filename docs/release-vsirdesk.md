# VsirDesk 发布说明

## 版本重点

- 将桌面小工具名称统一为 VsirDesk。
- 新增内置硬件监视模块，放在天气时间和快捷方式之间。
- 硬件监视模块显示 CPU 使用率、CPU 温度、内存使用率、GPU 使用率和 GPU 温度。
- 硬件监视模块跟随主程序主题、透明度、面板宽度和收起展开状态。
- 调整硬件监视模块高度，避免温度文字显示不全。
- 移除之前的独立硬件监视小窗方案，改为主程序内集成。

## 运行说明

解压发布包后运行 `LumiDesk.exe`。如果系统未安装 .NET 9 Desktop Runtime，需要先安装运行时。
